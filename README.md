# Frontend - Item Manager Lab Test

## Setup
1. Open a terminal inside the frontend folder.
2. Run:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open the local URL shown in the terminal, usually:
   ```text
   http://localhost:5173
   ```

## API URL
The frontend uses this API URL by default:
```text
http://localhost:5000/api
```

For live deployment, add this environment variable in Netlify or Vercel:
```text
VITE_API_URL=https://your-backend-live-url/api
```
